## 🔹 GitHub Project Description (Professional)

**AI & Cybersecurity Portfolio — Next.js Developer Portfolio**

A modern, high-performance personal portfolio built with **Next.js, TypeScript, and Tailwind CSS**, showcasing my work across **Artificial Intelligence, Cybersecurity, and Data Analysis**.

This portfolio is designed with a unique **multi-domain “Lens System”**, allowing users to dynamically switch between AI, Cybersecurity, and Data Analysis themes — each reflecting a different aspect of my technical expertise.

It highlights real-world, hands-on experience in:

* Network security and threat detection
* SOC (Security Operations Center) workflows
* Machine learning and AI-based anomaly detection
* IoT and embedded systems security

The project emphasizes **clean UI/UX design, accessibility (WCAG-compliant), responsive layouts, and production-level engineering practices**, including performance optimization and SEO readiness.

### Key Features

* ⚡ Next.js 15 App Router architecture
* 🎯 Dynamic Lens-based UI (AI / Cybersecurity / Data Analysis)
* 🌙 Dark/Light mode support
* 📱 Fully responsive and mobile-optimized design
* 🎨 Minimalist, modern UI inspired by top tech portfolios
* 🚀 Optimized performance and SEO
* 🧠 Real project experience integration from AI & Cybersecurity domains

### Tech Stack

Next.js · TypeScript · Tailwind CSS · Framer Motion · shadcn/ui · Lucide Icons
